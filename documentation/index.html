Folder Arrange

Unzip the archive and find out the folder "buyer-file". There are all the template files in this folder. You can open the following files for viewing in browser.

  1. dist — folder with final build.
  2. src/assets/style.css — css file.
  3. src/assets/images — folder with image files.
  4. src/assets/js/* — js file.
  5. src/partials/ — Partials.
  6. index.html Index file
  7. **.html Others Html files.
  8. index.js Webpack Project Entry file.
  9. .gitignore config for ignore git
  10. webpack.config.js Webpack configuration file.
  11. tailwind.config.js Tailwind configuration file.
  12. package.json Node.js project configuration.

Installation

  • Open CMD / Terminal in and open any template from all-templates folder and run npm install for installing necessary dependencies
  • Make sure you have installed nodejs on your system

Start Dev Server

  • Run npm run dev for running this project on your machine locally.

Build Project

  • Run npm run build for make a production build for this project. it will create a folder named /dist and final output will be there.

Customization

Change Theme Color

  • To change the theme color, open the tailwind.config.js file located in the root directory of your project.
  • In the tailwind.config.js file, you will find a section titled "Colors".
  • Modify the values of the colors you wish to change to meet your desired theme.

Change Font

  • Open the style.css file located in the root directory of your project.
  • In the style.css file, you can change the font of the site.

Change Logo

To change logo in the navigation panel, find image public/images/home-one/logo.png and replace it with your logo.

Note: Keep your logo inside the public/images/home-one folder.

FTP Upload

  • Open Your FTP manager and connect to your hosting
  • Browse to required directory(Normally index.html)
  • Upload the files inside dist folder
